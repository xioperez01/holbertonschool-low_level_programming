# 0x18. C - Dynamic libraries :green_book: :computer: :books: #
## Learning Objectives ##
### General ###
1. What is a dynamic library,
* how does it work,
* how to create one,
* and how to use it

3. What are the differences between static and shared libraries
4. Basic usage nm, ldd, ldconfig
